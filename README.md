## Projects

**[EchoPlay](https://github.com/huseyincancalti/EchoPlay)** · *Featured*
Windows video player purpose-built for NVIDIA ShadowPlay, NVIDIA App, and OBS dual-track recordings. Mixes game, microphone, and system audio live with zero re-encoding — solving a gap that no existing consumer player addresses.

**[XenUpdate](https://github.com/huseyincancalti/XenUpdate)** · *Featured*
Unified update manager for Windows, built on .NET 8 / WPF. Consolidates the Microsoft Store, `winget`, Windows Update, Device Manager, and GPU/BIOS driver updates into a single interface, with guided workflows for the updates that can't be automated.

**[Audio-Engine](https://github.com/huseyincancalti/Audio-Engine)**
Browser extension (Chrome / Firefox) for audio control: volume boost up to 1000%, 5-band EQ, dynamic range compression, and a per-domain rule engine that applies settings automatically.

**[TabGroup-Master](https://github.com/huseyincancalti/TabGroup-Master)**
Hierarchical tab and workspace manager for Chrome. Combines nested tree manipulation with a compiled Python macro engine via the Native Messaging API to bypass browser session-data limits.

**[Game Systems Showcase](https://karakedi.itch.io/)**
Playable prototypes demonstrating C# architecture patterns, custom state machines, and performance-focused gameplay mechanics.
